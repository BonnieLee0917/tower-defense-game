## 一、测试范围

本文档覆盖极速 MVP 版本的验收测试，聚焦 **核心游戏循环能跑通**。

### MVP Scope
- 1 张地图（20×11 grid, 64×64 tile, 12 个建塔点）
- 2 种塔：弓箭塔（70g, 单体）、炮塔（125g, AoE）
- 2 种敌人：普通兵（蓝色, 100hp, 80speed）、快速兵（黄色, 60hp, 140speed）
- 5 波递增难度
- 基础经济：200 初始金，击杀奖励
- HUD：生命值 / 金币 / 波次 / Next Wave
- 胜利 / 失败 + 重玩

## 二、测试 Checklist

### P0 — 核心循环（Must Pass）

#### 🗺️ 地图渲染
- [ ] 游戏加载后地图正确渲染，绿色草地 + 棕色路径
- [ ] 1280×720 分辨率下地图完整显示，无裁切
- [ ] 64×64 tile 网格对齐，无缝隙 / 重叠
- [ ] 12 个建塔点白色方框正确显示，位置不在路径上
- [ ] 路径从左侧(0, 224)出发，经 10 个 waypoint 到右侧(1280, 288)

#### 🚶 路径 & 敌人移动
- [ ] 敌人从地图左侧生成，沿 waypoint 路径完整走到右侧
- [ ] 路径转弯处移动平滑，无抖动 / 跳跃
- [ ] 普通兵（蓝色方块）和快速兵（黄色方块）视觉可区分
- [ ] 快速兵移动速度明显快于普通兵（140 vs 80）
- [ ] 敌人头顶血条正确显示（绿→黄→红）
- [ ] 多个敌人同时在路径上移动时无异常堆叠

#### 🏹 防御塔
- [ ] 点击空建塔点弹出建塔菜单（🏹 70g / 💣 125g）
- [ ] 金币充足时按钮高亮可点击，不足时灰色不可点击
- [ ] 建造弓箭塔后扣除 70g，建塔点变为弓箭塔（绿色圆 + "A"）
- [ ] 建造炮塔后扣除 125g，建塔点变为炮塔（红色圆 + "C"）
- [ ] 已建塔位置不可重复建塔（点击无反应）
- [ ] 弓箭塔向范围内最近敌人发射弹道，命中扣血
- [ ] 炮塔发射弹道命中后产生 AoE 溅射伤害（60px 范围）
- [ ] 鼠标 hover 塔显示攻击范围圆圈

#### 💰 经济系统
- [ ] 初始金币 200，HUD 显示正确
- [ ] 建塔后金币减少，HUD 即时更新
- [ ] 击杀普通兵 +5g，击杀快速兵 +7g
- [ ] 金币不足时无法建塔（按钮灰色）

#### 🌊 波次系统
- [ ] Wave 1 自动开始，生成 8 个普通兵
- [ ] 每波结束后显示 "Next Wave" 按钮
- [ ] 点击 "Next Wave" 触发下一波
- [ ] 5 波配置递增难度（后面波次敌人更多）
- [ ] HUD 波次显示正确（Wave 1/5 → 5/5）

#### ❤️ 生命值 & 胜负
- [ ] 初始生命值 20，HUD 显示正确
- [ ] 敌人走到终点 → 生命值 -1，HUD 更新
- [ ] 生命值归零 → 显示 "💀 Defeat!" + Retry 按钮
- [ ] 5 波全部清完 → 显示 "🎉 Victory!" + Play Again 按钮
- [ ] 点击 Retry / Play Again 重新开始游戏，状态完全重置

---

### P1 — 不崩溃（Should Pass）

#### 🔒 稳定性
- [ ] 全程游戏无白屏 / console error / 卡死
- [ ] 快速连续点击建塔不出重复建塔 / 金币异常
- [ ] 快速连点 "Next Wave" 不触发多波同时生成
- [ ] 游戏结束后点击地图无异常响应
- [ ] Retry / Play Again 后所有状态正确重置（金币、生命值、塔、敌人清空）
- [ ] 窗口缩放后游戏不崩溃（Scale Manager 自适应）

#### 🎯 边界情况
- [ ] 不建任何塔，20 个敌人走完 → 正确触发失败
- [ ] 全部建塔点占满后无法再建塔
- [ ] 同时大量敌人在屏（Wave 5: 25 个敌人）性能无明显卡顿

## 五、Phase 2 测试矩阵（已锁定方向）

> 基于 2026-04-07 已锁定产品方向：KR 经典模式、第一波手动开始、后续 15s 自动倒计时、启动资金 350、前低后高难度、Phase 2 基础移动兼容

### 1. 塔系统
- [ ] 4 种塔全部可建造：弓箭 / 法师 / 炮塔 / 兵营
- [ ] 各塔基础属性正确：费用 / 攻速 / 伤害 / 射程 / 特殊效果
- [ ] 3 级升级链路完整，升级后数值递增正确
- [ ] 法师塔对重甲目标的克制关系正确（以最终 PRD 为准）
- [ ] 兵营单位可正常生成、阻挡、死亡、重生
- [ ] 已建塔位可升级 / 出售，不可重复建造
- [ ] 出售返还 60% 总投入，含升级成本

### 2. 敌人系统
- [ ] 4 种敌人全部生效：普通 / 快速 / 重甲 / 飞行
- [ ] 普通兵：基础敌人，无异常属性
- [ ] 快速兵：速度明显快于普通兵
- [ ] 重甲兵：高血或减伤机制正确
- [ ] 飞行单位：路径 / 受击规则符合设计
- [ ] 敌人视觉区分明确，不会混淆

### 3. 波次与节奏
- [ ] 第一波前存在准备阶段，玩家可先建塔再开波
- [ ] 第一波需手动点击开始
- [ ] 后续波次在 15s 倒计时后自动启动
- [ ] 倒计时期间可提前发波，并正确获得 bonus
- [ ] 最后一波结束后不再触发下一波倒计时
- [ ] 完整波次扩展到 10-12 波后，整体难度前低后高

### 4. 经济与策略
- [ ] 启动资金 350 baseline 生效
- [ ] 击杀奖励按敌人类型正确发放
- [ ] 建塔 / 升级 / 出售 / 提前发波 bonus 全链路金额正确
- [ ] 金币不会出现负数、重复结算或丢结算
- [ ] 前 1-2 波体验友好，可完成基础教学与布防

### 5. 基础移动兼容（Phase 2）
- [ ] 手机横屏下画面完整显示，不变形、不裁切关键 UI
- [ ] HUD 在常见手机尺寸下不重叠、不错位、文字可读
- [ ] 主要交互可点：建塔点 / 建塔菜单 / 开波按钮 / 提前发波按钮
- [ ] 竖屏时有明确提示（旋转设备或兼容提示）
- [ ] iOS Safari / Android Chrome 基础兼容通过
