# Tower Storm Phase 3 — 视觉设计规格

> 作者：Vivian（Designer）
> 日期：2026-04-08
> 目标：为 Phase 3 所有新功能提供可直接落地的视觉参数

---

## 1. 全局技能 UI/特效

### Rain of Fire（Kane 负责开发）

**HUD 按钮：**
- 位置：HUD 右侧，Next Wave 按钮左边
- 图标：🔥 或红色圆形按钮
- 按钮底色：`0xD32F2F`（深红），圆形 r=20px
- 冷却中：灰化 `0x616161` + 扇形遮罩显示剩余 CD
- 可用时：脉冲动画（scale 1.0↔1.05，周期 1s）

**目标选择圈：**
- 点击按钮后进入选择模式
- 鼠标/手指位置显示目标圈：`strokeCircle` r=80px，色 `0xFF5722` alpha 0.5
- 内填充：`0xFF5722` alpha 0.1

**命中特效：**
- 延迟 0.5s 后触发
- 红色圆形扩散：从 r=10 扩展到 r=80，duration 300ms
- 颜色：`0xFF3D00` alpha 从 0.8 到 0
- 可选：3-5 个小火焰粒子向外扩散

### Reinforcements（Kane 负责开发）

**HUD 按钮：**
- 位置：Rain of Fire 按钮左边
- 图标：⚔️ 或蓝色圆形按钮
- 按钮底色：`0x1565C0`（深蓝），圆形 r=20px
- 冷却/可用状态同 Rain of Fire

**目标选择圈：**
- 只能放在路径上（非路径区域显示禁止标识）
- 路径上显示：`strokeCircle` r=40px，色 `0x42A5F5` alpha 0.5

**士兵出现特效：**
- 3 个临时士兵在目标点三角布阵出现
- 出现时 scale 从 0 弹到 1（Back.easeOut 200ms）
- 15s 后闪烁 3 次（alpha 0↔1）然后消失

## 2. Lv4 双专精塔视觉

### 通用规则
- Lv4 专精塔比 Lv3 明显更大（底座+6px）
- 有独特的颜色主调区分两个分支
- 有专属标识元素

### 弓箭塔专精

**Rangers Lodge（远程狙击）：**
- 主色：`0x2E7D32`（深森林绿）
- 标识：塔顶加瞄准镜图标（十字线）
- 体型：比 Lv3 更高更瘦
- 底座：木质底座 `0x5D4037`

**Musketeer Garrison（爆发 DPS）：**
- 主色：`0xE65100`（橙红）
- 标识：塔顶加火焰图标
- 体型：比 Lv3 更粗壮
- 底座：金属底座 `0x78909C`

### 兵营专精

**Holy Order（坦克骑士）：**
- 主色：`0xFFD600`（金色）
- 标识：十字盾牌
- 士兵形象：更大 + 大盾
- 旗帜色：白色 `0xFFFFFF`

**Assassin Guild（高速刺客）：**
- 主色：`0x4A148C`（暗紫）
- 标识：毒药瓶 / 匕首
- 士兵形象：更小更快 + 双匕首
- 旗帜色：紫色 `0x7B1FA2`

### 魔法塔专精

**Arcane Wizard（AoE 核弹）：**
- 主色：`0x1A237E`（深蓝）
- 标识：大能量球（比 Lv3 更大更亮）
- 能量球色：`0x448AFF`
- 底座椭圆：`0x0D47A1`

**Sorcerer（群控减速）：**
- 主色：`0x00695C`（深青）
- 标识：旋转符文环
- 符文色：`0x00E5FF`
- 底座椭圆：`0x004D40`

### 炮塔专精

**Big Bertha（超大 AoE）：**
- 主色：`0xB71C1C`（暗红）
- 标识：超粗炮管（width 16px）
- 底座更宽：width+8px
- 炮口火焰：`0xFF6F00` 常驻

**Tesla（闪电链）：**
- 主色：`0x0277BD`（电蓝）
- 标识：塔顶电球 + 闪电弧
- 电球色：`0x4FC3F7`
- 闪电链：`0x00E5FF` 锯齿线连接目标

## 4. 星级评价 UI

- 位置：结算画面标题下方
- 3 颗星排列，间距 16px
- 获得的星：`0xFFD600` 金色 + 描边 `0xFFA000`
- 未获得的星：`0x424242` 灰色 + 描边 `0x212121`
- 星星大小：32px
- 动画：每颗星依次弹出（scale 0→1.2→1.0，间隔 300ms）

评价标准（写在结算规格里了）：
- 18+ 生命 → ⭐⭐⭐
- 6-17 生命 → ⭐⭐
- 1-5 生命 → ⭐

## 5. 2x 加速 / 暂停 UI

**按钮位置：** HUD 右侧区域
- ⏸️ 暂停按钮：`0x78909C`（灰色）圆形 r=16px
- ⏩ 加速按钮：`0x1976D2`（蓝色）圆形 r=16px
- 激活时（2x 模式）：底色变 `0xFF8F00`（橙色），文字显示 "2x"

**暂停遮罩：**
- 全屏 `0x000000` alpha 0.5
- 居中大字 "PAUSED" `0xFFFFFF` 48px
- 点击任意处或再点暂停按钮恢复

---

## 6. 波次预览

**位置：** 每波开始前的倒计时期间，在地图下方或 HUD 区域显示
**内容：** 下一波的敌人类型 + 数量
**格式：**
```
Next: 🧟×5  🐺×3  🛡️×2
```
- 背景：`0x263238` alpha 0.8，圆角 8px
- 文字：`0xFFFFFF` 14px
- 自动在波次开始后消失
