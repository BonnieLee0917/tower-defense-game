## 一、设计方向

**风格关键词：** 扁平卡通、轻手绘感、明快但不刺眼、信息清晰优先

**参考方向：** Kingdom Rush 的色彩饱和度 + 现代扁平 UI 的简洁感

**设计原则：**
- 功能色 > 装饰色（每种颜色都有明确用途）
- 高对比度保证可读性（塔/敌人/路径/草地需要一眼区分）
- 色板尽量控制在 20 色以内，保持视觉统一

## 三、Tile 规格

### 3.1 基础参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 画布分辨率 | 1280 × 720 px | 基准分辨率，Scale Manager 自适应 |
| Tile 尺寸 | 64 × 64 px | 地图网格单元 |
| 地图网格 | 20 列 × 11 行 | 1280/64 × 704/64（顶部 16px 给 HUD） |
| 可用地图区域 | 1280 × 704 px | y: 16 ~ 720 |
| HUD 区域 | 1280 × 40 px | y: 0 ~ 40，覆盖在地图上层 |

### 3.2 Sprite 尺寸规范

| 元素 | 画布尺寸 | 可绘制区域 | 说明 |
|------|---------|-----------|------|
| 塔（含底座） | 64 × 64 px | 56 × 56 px 内 | 留 4px padding 防止贴边 |
| 塔主体 | — | 半径 18-22 px | 圆形/近圆形 |
| 塔底座 | — | 48 × 48 px | 方形灰色底 |
| 敌人 | 32 × 32 px | 24 × 24 px 内 | 小于塔，保持视觉层级 |
| 弹道 | — | 半径 3-6 px | 圆形 |
| 建塔点标识 | 56 × 56 px | — | 比 tile 略小 |

### 3.3 血条规范

| 参数 | 值 |
|------|-----|
| 宽度 | 30 px |
| 高度 | 4 px |
| 距头顶 | 8 px |
| 圆角 | 0（当前），建议后续改 2px |

## 四、Haaland 现有代码 → 色板映射

以下是当前代码中的颜色和建议替换对照，方便 Haaland 快速更新：

### gameConfig.ts

```typescript
// 弓箭塔
color: 0x4CAF50,          // 原 0x00cc00
projectileColor: 0xCDDC39, // 原 0xccff00

// 炮塔
color: 0xE64A19,          // 原 0xcc0000
projectileColor: 0x455A64, // 原 0x333333

// 普通兵
color: 0x42A5F5,          // 原 0x3366ff

// 快速兵
color: 0xFFCA28,          // 原 0xffcc00（接近，微调更柔和）
```

### GameScene.ts — drawMap()

```typescript
// 草地
gfx.fillStyle(isPath ? 0xC4956A : 0x3A7D44, 1);
// 网格线
gfx.lineStyle(1, 0x000000, 0.10);
```

### GameScene.ts — drawBuildSpots()

```typescript
gfx.lineStyle(2, 0xF0E6D3, 0.6);
gfx.fillStyle(0xF0E6D3, 0.15);
```

### GameScene.ts — createHUD()

```typescript
// Next Wave 按钮
backgroundColor: '#1976D2'  // 原 '#2266aa'
```

### GameScene.ts — showBuildMenu()

```typescript
// 菜单背景
bg.fillStyle(0x263238, 0.92);  // 原 0x222222, 0.9

// 弓箭塔按钮文字色
color: archerAfford ? '#4CAF50' : '#616161'  // 原 '#00ff00' : '#666666'

// 炮塔按钮文字色
color: cannonAfford ? '#E64A19' : '#616161'  // 原 '#ff4444' : '#666666'
```

### BaseTower.ts — draw()

```typescript
// 塔底座
this.gfx.fillStyle(0x78909C, 1);  // 原 0x888888
```

### BaseEnemy.ts — draw()

```typescript
// 血条背景
this.hpBar.fillStyle(0x37474F, 1);  // 原 0x333333
// 血条颜色
const hpColor = ratio > 0.5 ? 0x4CAF50 : ratio > 0.25 ? 0xFF9800 : 0xF44336;
// 原: 0x00ff00 : 0xffaa00 : 0xff0000
```
